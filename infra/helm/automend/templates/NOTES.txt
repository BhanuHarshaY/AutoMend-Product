Frontend + API
--------------
  URL:  http://{{ .Values.ingress.host }}/

  If running on kind / minikube, add to /etc/hosts first:
    127.0.0.1  {{ .Values.ingress.host }}

  Or port-forward the Ingress controller:
    kubectl port-forward -n ingress-nginx svc/ingress-nginx-controller 8080:80
    # Then open http://localhost:8080/ (set Host header via browser extension)
{{- else }}

Ingress is disabled. Reach the services directly via port-forward:

  # Frontend
  kubectl port-forward -n {{ .Release.Namespace }} \
    svc/{{ include "automend.componentFullname" (dict "ctx" . "component" "frontend") }} \
    3000:{{ .Values.frontend.service.port }}

  # API
  kubectl port-forward -n {{ .Release.Namespace }} \
    svc/{{ include "automend.componentFullname" (dict "ctx" . "component" "api") }} \
    8000:{{ .Values.api.service.port }}

Then open:
  Frontend:  http://localhost:3000
  API docs:  http://localhost:8000/docs
{{- end }}

      python scripts/bootstrap_admin.py
