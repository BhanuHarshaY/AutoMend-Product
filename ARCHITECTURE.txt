================================================================================
Data Flow Summary
================================================================================

Runtime (logs → remediation):
  Workload Pods → Fluent Bit → OpenTelemetry Collector → Redis stream `normalized_logs`
    → WindowWorker (5-min windows) → RoBERTa-base classifier (:8001)
    → Redis stream `classified_events` + classifier_outputs (Postgres)
    → CorrelationWorker (dedup, cooldown, TriggerRule lookup, kill-switch check)
    → Incident row (Postgres) + Temporal DynamicPlaybookExecutor start
    → Activities execute against Kubernetes API / Prometheus / Slack
    → BroadcastService → Redis Pub/Sub → WebSocket → Browser

Design-time (intent → playbook):
  Browser (Playbook Builder) → POST /api/design/generate_workflow
    → ArchitectClient → Qwen 3.5 vLLM → PlaybookSpec JSON
    → adapters.specToReactFlow → ReactFlow canvas
    → user edits → adapters.reactFlowToSpec
    → POST /api/playbooks/{id}/versions → EmbeddingService → pgvector
    → TriggerRule binds incident_type → PlaybookVersion
